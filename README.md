## Escopo do projeto

  ### Definição
  Desenvolver um interpretador de comandos no ambiente linux.
  
  ### Comandos
    * mudar   (cd) 
    * local   (pwd)
    * apagar  (rm)
    * criar   (mkdir)
    * remover (rmdir)
    * copiar  (cp)
    * data    (data )
